# 配图来源清单

本文件定义文章配图的来源优先级和操作规范。

用户主动询问配图建议时才启用，不在默认流程中。

## 来源优先级

1. **Wikimedia Commons**（首选）
   - 公版可商用
   - 覆盖历史、科学、人物、地理等各类主题
   - 搜索：https://commons.wikimedia.org/

2. **NASA**
   - 公共领域，可商用
   - 适合：太空、天文、地球、航空主题
   - 搜索：https://images.nasa.gov/

3. **政府与公共机构**
   - 美国政府机构（NOAA、USGS、NIH 等）的图片大多公共领域
   - 欧盟、联合国公开资料也可用
   - 使用前确认具体许可

4. **学术论文**
   - 开放获取期刊（arXiv、PLOS、Nature Open）的图可引用
   - 必须标注来源和作者

## 操作规范

- 用 WebSearch 查找具体图片和链接，**分章节列出**
- 每张图说明：
  - 图片描述
  - 来源（网站名 + 作者/机构）
  - URL
  - 许可证（如 CC BY 4.0、Public Domain）
- **不要把图片插入正文**，只列清单让用户自己挑选
- 主动推荐"高 ROI"组合：5 张以内能撑起一篇图文稿的搭配

## 禁区

- 不用未标明许可的图片
- 不用商业图库（Shutterstock、Getty）链接，除非用户明确说有授权
- 不用 AI 生成图片冒充真实资料图
